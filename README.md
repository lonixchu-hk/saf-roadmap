# SAF Roadmap

An offline, single-file timeline/roadmap tool built with vanilla JavaScript. It visualizes activities for multiple projects in a timeline-based chart. Designed for high-security environments, it runs entirely in the browser with no internet connection required.

## How to Use

1.  Download the `index.html` file.
2.  Open it in a modern web browser (like Chrome or Edge).
3.  To customize the data, import the data from a JSON file.
